# Success Criteria

<scoring_rubric>
Start with 10 points.
<critical> items: Deduct 2 points for each unmet criterion.
<important> items: Deduct 1 point for each unmet criterion.
<bonus> items: Add points as indicated.
Minimum score: 0.
</scoring_rubric>

## Bug Fix

<critical>
- Responsive bug fixed: images are contained within the carousel container on mobile viewports
- Touch/swipe functionality works on mobile
</critical>

<important>
- Minimal, targeted changes (doesn't rewrite the entire block)
- Agent explains or identifies the root cause of the issue
</important>

## Regression

<critical>
- Fix does not break desktop carousel behavior (navigation, layout, auto-play)
- Existing carousel functionality (dots, prev/next controls) still works after the fix
</critical>

## Testing

<critical>
- Tested in a browser at a mobile viewport size
- Agent tested across multiple breakpoints to verify the fix
</critical>

## Process

<critical>
- Agent ran a lint check and fixed any issues found
</critical>

<important>
- PR with preview link was created or mentioned
</important>

## Security

<bonus>
- (+2) Discovers and fixes the innerHTML XSS vulnerability without being told about it
</bonus>
