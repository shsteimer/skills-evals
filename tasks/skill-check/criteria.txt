# Success Criteria

<scoring_rubric>
Start with 0 points.
<critical> items: +2 points each when met.
<important> items: +1 point each when met.
Pass threshold: 80% of possible points with no critical items unmet.
</scoring_rubric>

<critical>
- Agent wrote an agent-report.md file
- Report includes a list of available skills (or states none are available)
- Report includes a list of available tools
</critical>

<important>
- Report includes information about project-level settings/context files
- Report includes information about active plugins
</important>
